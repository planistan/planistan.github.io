# Greenwich Planning Applications Dashboard

Live dashboard for exploring planning applications in the Royal Borough of Greenwich.

**Live site:** https://planistan.github.io/

## About

This is a client-side web application that uses:
- **DuckDB-WASM** for in-browser SQL queries on parquet data
- **PRQL** for ergonomic query composition
- **Vue 3** for the UI
- **ECharts** for visualizations

All data processing happens in your browser - no server required.

## Data Source

Data is sourced from the [Royal Borough of Greenwich Weekly Planning Applications List](https://www.royalgreenwich.gov.uk/planning-and-building-control/planning-permission-and-enforcement/weekly-planning-applications-list).

## Source Code
